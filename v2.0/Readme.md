+ 登录模块：ok
+ 数据库模块：连接Mysql，提供一些方法
+ 文件夹模块：ok
+ 下载模块，百分比下载
+ 接口模块：Flask框架，提供随机作品，作品总数，特定作品信息
+ 爬虫解析模块：返回需要下载的画师的
